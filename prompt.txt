## TON TRAVAIL

Crée une **application web standalone** avec **React 18 + React Flow** qui permet de :

### 1. VISUALISER le flow
- Afficher tous les nœuds du JSON sous forme de graphe interactif
- Utiliser React Flow pour le rendu visuel
- Colorer les nœuds selon leur type :
  - 🟦 **Bleu** : type "menu"
  - 🟩 **Vert** : type "filter"  
  - 🟨 **Jaune** : type "results"
  - 🟪 **Violet** : type "widget"
- Afficher les connexions entre nœuds (flèches) basées sur :
  - `option.next` (pour les options)
  - `next_filter` (pour les filtres)

### 2. ÉDITER les nœuds
Quand on clique sur un nœud, afficher un **panel latéral droit** avec tous les champs éditables :

**Section Général**
- Input : `id` (lecture seule)
- Input : `label` (à ajouter au JSON si absent)
- Select : `type` (menu | filter | results | widget)
- Number : `level`

**Section Audio**
- Liste des fichiers audio contexte (`audio.context[]`)
- Bouton "+ Ajouter audio"
- Checkbox : `audio.auto_play`

**Section Options** (si type = menu ou filter)
- Liste éditable des options avec pour chaque :
  - Input : `id`
  - Number : `number` (1-5)
  - Input : `label`
  - Input : `image` (chemin)
  - Liste : `audio[]` (fichiers)
  - Tags : `keywords[]`
  - Select : `next` (dropdown tous les nœuds)
  - JSON editor : `params`
  - Bouton : 🗑️ Supprimer
- Bouton : **+ Ajouter option**

**Section Pagination**
- Number : `items_per_page`
- Checkbox : `allow_swipe`

**Section Navigation**
- Checkbox : `show_back`
- Checkbox : `show_home`
- Tags éditables : `breadcrumb[]`

**Section API** (si présent)
- Input : `endpoint`
- Select : `method` (GET | POST)
- Checkbox : `params_from_history`

**Section Filtres** (si type = filter)
- Select : `filter_category`
- Input : `filter_step`
- Select : `next_filter` (dropdown nœuds)

### 3. AJOUTER de nouveaux nœuds
- Bouton flottant "+ Nouveau nœud"
- Modal avec choix du type :
  - Menu
  - Filter
  - Results
  - Widget
- Créer nœud avec structure par défaut selon le type
- Placer visuellement sur le canvas

### 4. CONNECTER les nœuds
- Drag & drop pour créer connexions entre nœuds
- Mise à jour automatique du JSON (`next`, `next_filter`)

### 5. SUPPRIMER nœuds/connexions
- Sélectionner nœud → touche Delete → Supprime nœud
- Clic sur connexion → Bouton supprimer → Supprime lien

### 6. SAUVEGARDER / CHARGER
- Bouton **💾 Télécharger JSON** : Exporte le flow édité
- Bouton **📂 Charger JSON** : Upload un fichier JSON
- Auto-save dans localStorage toutes les 30 secondes

### 7. VALIDER le flow
- Bouton **✅ Valider**
- Vérifications :
  - ❌ Nœuds avec `next` pointant vers ID inexistant → Erreur
  - ⚠️ Nœuds orphelins (aucun nœud ne pointe vers eux, sauf root) → Warning
  - ⚠️ Nœuds sans options (si type menu/filter) → Warning
- Afficher résultats validation en haut de l'écran

## GESTION DES ERREURS

- **Fichier JSON invalide** : Message d'erreur clair + garde l'ancien flow
- **Nœud sans ID** : Génère ID automatique `node_${timestamp}`
- **Option sans next** : Pas d'erreur, juste warning
- **Connexion vers ID inexistant** : Erreur rouge, supprime connexion
